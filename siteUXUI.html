User Experience & User Interface

Good design is how something looks, feels and works.

UX

brugergrænseflæde

  • How Effective?
  • Functional elements
  • Understanding the problem
  • Sketching, wireframing & prototyping
  • Usability testing
  • User flow

Målet er at forstå brugeren og hans behøv for at definiere og løse problem; skabe et produkt, der er nemt ved brug. The 4Cs of UX design: Consistency, Continuity, Context and Complementary = human-centric approach.

UI

brugeroplevelse

  • How pleasent?
  • Visual design & aesthetic impression
  • Content layout, graphical user interfaces
  • Colors, typography, icons, pictures
  • Buttons style, menu
  • Responsiveness, voice-controlled interfaces

UI is the point of contact between humans and computers. Målet er at lave et produkt, der ser godt ud og har en interessant og legende brugrænseflade .

Opsummering:

Den væsentligeste forskel mellem UI og UX er, hvad de fokuserer på, og hvordan brugeren interagerer med dem. I stedet for, at stille UX vs UI , bor vi tale om Ux og UI fordi de overlapper og supplerer hinanden.
