Design Systems within UI

A design system is a collection of reusable UI components. In it, you can define UI elements such as buttons as well as text elements and other elements that you're sure will be reused throughout the design file fx build the color palette. This makes the design workflow very easy.UI kit focuses on giving designers the visual templates needed to create new pages or views, a design system is much more advanced
