Minimalism

  • eliminating excess, keeping strictly necessary
  • functionality for every element
  • strict visual hierarchy
  • simplicity and clarity
  • white space incorporatation
  • attention to proportions and compositions
  • neutral colors
  • monochromatic palette with colour used as an accent
  • strategically placing of elements
  • small object occupy large spaces
  • typography is significant

Augmentet Reality

  • AR blends the physical and digital world. It´s an interactive experience that enhances the real world with computer-generated perceptual information;
  • Real-world related: users can interact with virtual objects in the real world:for example, consumers can use AR to try on different pairs of sunglasses or see how a sofa would fit within their home.
  • real-time animations based on the user's interactions with the app;
  • use real-time face filter;
  • lets user create own AR avatar that is able to look and talk exactly like the user;
  • 3D UI:three-dimensional user interfaces allow users to interact with virtual objects, environments;

Gradients

  • gradient is the gradual blending from one color to another. It enables the designer to almost create a new color;
  • add depth and dimension to the design;
  • makes objects stand out by adding a new dimension;
  • adds realism to the object;
  • can be bold or subtle, also a focal point of the design
  • creates a more visually interesting and dynamic eye-catching look;

Neuromorphism

  • subtle yet innovative look;
  • illusion of extruded shapes
  • it place the user interface elements behind the background - similar to the watermark setting. As soon as the user selects that element, it protrudes out and looks like it is coming out of the screen.
  • keeps the background and UI elements' colors the same and uses shadow,light, highlights, glow;
  • make messages clearer , do interactions easier and make interface more stylish;
